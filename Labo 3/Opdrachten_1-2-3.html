Opdracht 1

Waarmee komen de blokjes overeen?

De blokjes komen overeen met de volgorde waarin de code geschreven wordt. Het bovenste is het eerste dat wordt geschreven in de code en zo gaan we naar beneden, als iets op dezelfde hoogte staat als een ander betekent het dat deze ook in de code op dezelfde hoogte staan geschreven.

En de lijntjes ertussen?

De lijntjes tussen de blokken geven aan welke blok er in een andere blok hoort. Zo horen de meta en de title in de head blok en de head en body in de html blok, enzovoort.

Opdracht 3

Uit welke elementen bestaat elke accordion section in de DOM-tree?

Een h3 tag, een div-element en een paragraaf.

Welke wijzigingen gebeuren er in de DOM-tree als je op een gesloten section klikt?

De eigenschappen van de div wijzigen. Bijvoorbeeld: De code 'display none' verdwijnt, en in de sectie die voorheen open stond verschijnt nu wel de 'display none'. De stijl van de header wordt nu active om de blauwe kleur te tonen. Aria-elementen wijzigen.

Hoe zou dit gebeurd zijn?

Door het openklikken van een andere sectie wordt een query uitgevoerd die de gewijzigde code aanroept.
