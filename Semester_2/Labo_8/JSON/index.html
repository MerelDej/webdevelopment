Welk formaat wordt er eigenlijk gebruikt om het Date object? ISO formaat: "yyyy-mm-ddThh:mm:ss.sssZ", waarbij de T voor "tijd" staat, m.a.w. vanaf daar start de tijd, en de Z staat voor de UTC-tijdsstempel.

Wat gebeurt er als eenzelfde object meermaals voorkomt bij het opbouwen van de JSON tekst? Het object wordt meermaals opgenomen in de JSON-string.
